# 如何新增DNS
## Windows 10使用者
1. 請先開啟windows terminal，執行```ipconfig /all```，查看目前連入Office網域所使用的網路連線(例如`乙太網路`)透過DHCP自動取得的DNS Servers，並先保留此windows terminal別關掉，後續步驟會用到。
2. 將滑鼠移至右下`網際網路存取`icon -> 按滑鼠右鍵 -> 點選`開啟網路和網際網路設定`
3. 在開啟的設定視窗中，找到`變更介面卡選項`並點選
4. 在開啟的網路連線視窗中，右鍵點選目前連入Office網域所使用的網路連線 -> 點選`內容`
5. 在開啟的內容設定視窗中，找到`網際網路通訊協定第4版(TCP/IPv4)`並點選 -> 再點選`內容`
6. 在開啟的內容設定視窗中，找到`進階`並點選
7. 在開啟的進階設定視窗中，點選`DNS`頁籤
8. 點選`新增` -> 輸入 `192.168.201.83` -> 按下`新增`
9. 重覆第8步驟，將第1步驟中所查看到透過DHCP自動取得的DNS Servers都新增進去
10. 確認`192.168.201.83`排在順序的最前
11. 按下`確定` -> 按下 `確定` -> 按下 `關閉`使設定生效。
12. 另開啟windows terminal，執行```ipconfig /all```，確認DNS Servers己修改至欲網改的網路連線中。
