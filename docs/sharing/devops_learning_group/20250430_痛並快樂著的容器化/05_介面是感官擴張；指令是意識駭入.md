# 本地開發環境&工具
  1. K8S叢集
     - [Minikube](https://minikube.sigs.k8s.io/docs/start/)：一個可在本地虛擬機或容器中快速啟動多節點K8S叢集的工具，適合初學者與功能測試。需搭配適當的[driver](https://minikube.sigs.k8s.io/docs/drivers/)
     - [K3d](https://k3d.io/stable/)：將輕量級的k3s叢集運行在Docker容器中，方便快速建立多節點叢集，適合進階本地開發與 CI/CD測試
     - [kind](https://kind.sigs.k8s.io/)：透過Docker容器模擬K8S節點，專為測試 Kubernetes元件與CI環境而設計，支援高度自訂的叢集配置
  2. K8S管理介面
     - [~~Lens~~](https://k8slens.dev/)：有年營收需小於1000萬美金的使用限制
     - [K9s](https://k9scli.io/)：文字為主的UI
     - Kubernetes VSCode plugin：VSCode IDE內的陽春UI
     - [~~Portainer CE~~](https://docs.portainer.io/start/install-ce)：不支援匯入kube config的方式連線至K8S。
