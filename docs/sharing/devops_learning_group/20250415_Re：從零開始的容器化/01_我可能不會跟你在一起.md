# 系統運行問題
 - 同一主機的每個系統需要的語言套件、系統函式庫(xxx.so)、設定都不相同
 - 同一主機的不同系統可能會讀寫相同目錄/檔案/環境變數
 - 同一主機的不同系統可能會開啟相同的port
 - 同一主機需要運行同一系統的不同版本
 - 系統在某台主機無法運行，但在其它主機都沒問題
